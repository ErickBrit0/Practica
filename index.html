Iniciandno con Html y su estructura

Estructura de HTML

La estructura HTML de una pagina web se compone de las etiquetas pirncipales que son:

  • html
  • head
  • body
indicando con esto, tanto el inicio de la pagina, la contencio de informacion mas descriptiva de la pagina y finalmente su conformacion bisible de esta.

Etiquetas de HTML

Las etiquetas html son el lenguaje utilizado para estructurar y definir el contenido en un elemento HTML estas etiquetas se encuentran en el HTML de cada pagina.

Atributos y Elementos

Un elemento es un conjunto completo de etiquetas y contendo que define un elemento especifico en una pagina web.

Los atributos proporcionan informacion adicional a los elementos HTML como se debe procesar o mostrar el contenido.
